API Documentation

INVESTHINK API

Integrate powerful crypto analysis and AI capabilities into your own applications

Get Your API Key

Sign up for an Enterprise account to get access to our API

Base URL

https://api.investhink.com/v1

Authentication

All API requests require authentication using an API key. Include your API key in the request header: 

Authorization: Bearer YOUR_API_KEY

Endpoints

GET /crypto/price/{symbol}

Get current price and market data for a specific cryptocurrency

Parameters

Name Type Required Description
symbol string Required Cryptocurrency symbol (e.g., BTC, ETH)

Response Example

{
  "symbol": "BTC",
  "name": "Bitcoin",
  "price": 67234.50,
  "change_24h": 2.45,
  "volume_24h": 2400000000,
  "market_cap": 1300000000000,
  "timestamp": "2024-12-01T12:00:00Z"
}
POST /ai/analyze

Get AI-powered analysis for a cryptocurrency

Request Body

Name Type Required Description
symbol string Required Cryptocurrency symbol
timeframe string Optional Analysis timeframe (1h, 4h, 1d, 1w)
indicators array Optional Technical indicators to include

Response Example

{
  "symbol": "BTC",
  "analysis": {
    "trend": "bullish",
    "signal": "buy",
    "confidence": 0.85,
    "entry_point": 66500,
    "target": 70000,
    "stop_loss": 65000,
    "indicators": {
      "rsi": 58,
      "macd": "positive",
      "ema_20": 66200,
      "ema_50": 65800
    }
  },
  "summary": "Strong bullish momentum with potential upside..."
}
GET /screener

Screen cryptocurrencies based on custom criteria

Query Parameters

Name Type Required Description
filter string Required Filter type (top_gainer, top_loser, high_volume)
limit integer Optional Number of results (default: 10, max: 100)

Response Example

{
  "filter": "top_gainer",
  "results": [
    {
      "symbol": "ETH",
      "name": "Ethereum",
      "price": 3567.00,
      "change_24h": 5.67,
      "volume_24h": 1800000000
    },
    {
      "symbol": "BNB",
      "name": "Binance Coin",
      "price": 421.00,
      "change_24h": 4.23,
      "volume_24h": 890000000
    }
  ]
}
POST /watchlist

Add cryptocurrency to watchlist

Request Body

{
  "symbol": "BTC",
  "note": "Strong support at 65000",
  "alert_price": 70000
}
